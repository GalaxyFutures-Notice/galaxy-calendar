# 📋 Galaxy Calendar 网站部署操作手册

## 目录结构说明

```
galaxy-calendar/               ← 这是您的整个网站文件夹
├── index.html                 ← 首页（月份导航）
├── vercel.json                ← Vercel配置文件（不要修改）
└── calendars/                 ← 存放每月日历的文件夹
    ├── 2026-04.html           ← 四月日历（已有）
    ├── 2026-05.html           ← 五月日历（下月添加）
    └── ...
```

---

## 第一步：创建 GitHub 账号和仓库（一次性，约10分钟）

1. 打开 https://github.com，点击 **Sign up** 注册账号
2. 登录后，点击右上角 **+** → **New repository**
3. 填写：
   - Repository name: `galaxy-calendar`
   - 选择 **Public**（免费托管必须公开）
   - 勾选 **Add a README file**
4. 点击 **Create repository**

---

## 第二步：上传网站文件（一次性）

1. 在您的仓库页面，点击 **Add file** → **Upload files**
2. 将整个 `galaxy-calendar` 文件夹里的**所有内容**拖入上传区域
   - `index.html`
   - `vercel.json`
   - `calendars/` 文件夹（含 `2026-04.html`）
3. 滚动到底部，点击 **Commit changes**

---

## 第三步：部署到 Vercel（一次性，约5分钟）

1. 打开 https://vercel.com，点击 **Sign Up**
2. 选择 **Continue with GitHub**（用GitHub账号登录）
3. 点击 **Add New Project**
4. 找到您的 `galaxy-calendar` 仓库，点击 **Import**
5. 所有设置保持默认，直接点击 **Deploy**
6. 等待约30秒，看到 **Congratulations** 即部署成功！
7. Vercel会给您一个网址，例如：
   `https://galaxy-calendar.vercel.app`

---

## 每月更新操作（日常工作，约2分钟）

### 第一步：命名新文件
将新月份的HTML文件按以下格式命名：
- 五月 → `2026-05.html`
- 六月 → `2026-06.html`
- 以此类推...

### 第二步：上传到 GitHub
1. 打开您的 GitHub 仓库
2. 点击进入 `calendars` 文件夹
3. 点击 **Add file** → **Upload files**
4. 上传新的 HTML 文件
5. 点击 **Commit changes**

### 第三步：更新首页配置（重要！）
1. 回到仓库根目录，点击 `index.html`
2. 点击右上角的铅笔图标（✏️ Edit this file）
3. 找到以下这段代码（约第166行）：

```javascript
const AVAILABLE = {
    "2026-04": "calendars/2026-04.html",
    // 下个月发布后，添加一行:
    // "2026-05": "calendars/2026-05.html",
};
```

4. 在花括号内添加新月份，例如五月：

```javascript
const AVAILABLE = {
    "2026-04": "calendars/2026-04.html",
    "2026-05": "calendars/2026-05.html",
};
```

5. 点击 **Commit changes**

### 第四步：Vercel自动发布
完成上述步骤后，Vercel会在 **约60秒内自动更新网站**，无需任何额外操作。

## 可选：绑定自定义域名

如果您希望链接显示为 `calendar.yourdomain.com`：
1. 在 Vercel 项目页面 → Settings → Domains
2. 输入您的域名，按提示在域名服务商处添加DNS记录
3. 约10分钟后生效

## 常见问题

**Q: 上传后网站没有更新？**
A: 等待1-2分钟，Vercel自动部署需要约60秒。

**Q: 首页显示某月是灰色不可点击？**
A: 需要在 `index.html` 的 `AVAILABLE` 配置中添加该月份。

**Q: 如何删除某个月的日历？**
A: 从 `AVAILABLE` 配置中删除对应行即可，文件本身可以保留。
